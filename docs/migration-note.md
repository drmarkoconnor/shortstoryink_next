# Migration Note

This project is no longer using Eleventy as the frontend framework.

Historical docs may still reference:

- Eleventy
- Nunjucks
- static-first templating assumptions
- Netlify Functions tied to the old frontend

Those references should be treated as historical implementation context only.

Current frontend direction:

- Next.js App Router
- React
- Tailwind CSS
- Supabase backend
- GitHub + Netlify deployment

Product vision, design direction, workflow goals, and domain model remain valid
unless explicitly replaced.
