# Image folders

Keep image assets organised by where they are used in the site. This helps
maintain a calm, predictable structure as the design evolves.

## Folders

- `reading-room/`
  - Atmospheric hero/background washes for Reading Room pages.
  - Usually larger, moody images that can sit behind typography.

- `workshop/`
  - Workshop-related imagery (submission/feedback/revision contexts).
  - Rarely needed in v1, but reserved for when we introduce comfortable,
    restrained visual cues.

- `studio/`
  - Studio/dashboard imagery (if ever used—keep minimal).

- `library/`
  - Library visuals (handout cover images, section dividers, etc.).

- `brand/`
  - Marketing/public site visuals.

- `misc/`
  - Temporary holding area for unsorted images.
  - Try not to leave images here long-term.

## Usage

Most templates reference images via the passthrough path:

- `/assets/images/...`

Example (Reading Room hero):

- `heroImage: /assets/images/reading-room/us-57.jpg`
